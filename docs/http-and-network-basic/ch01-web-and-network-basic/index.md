## 1.2 나의 사춘기에게

- 웹은 '지식 공유'에서 시작됐다.
- 1996년 5월, 'HTTP'가 등장했다.

## 1.3 모든 날, 모든 순간

- 네트워크의 기본은 'TCP/IP'다.
- 애플리케이션 계층, 전송 계층, 인터넷 계층, Network Access 계층으로 분류한다.
    - 변화에 유연하게 대응하기 위해 계층을 나눈다.

## ETC

- IP, TCP, DNS
- URI, URL

### URI vs. URL

URI(Uniform Resource Identifier)에 URL을 포함한다. 일상에서 자주 이야기하는 내용은 엄밀히 따지면 URL이다.
